# Unix-shell-in-C

### Implementation of a Unix shell in C. The instructions of bash that are implemented are the following:
+ "-depth"
+ "-gid", "-group"
+ "-help", "--help"
+ "-iname", "-inum", "-ignore_readdir_race"
+ "-maxdepth", "-mindepth", "-mount"
+ "-name", "-newer"
+ "-type"
+ "-uid", "-user"
+ "-version", "--version"
+ "-xdev"
